# E-Sports Tournament Management System

This is a Django-based web application for managing E-Sports tournaments.

## Features
-   Tournament Registration
-   User Profiles
-   Team Management
-   Match Scheduling
-   Admin Dashboard

## Getting Started
1.  Clone the repository.
2.  Install dependencies: `pip install -r requirements.txt`. (Note: create requirements.txt if not present)
3.  Run migrations: `python manage.py migrate`.
4.  Start the server: `python manage.py runserver`.
